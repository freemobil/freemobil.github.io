It's time to build for tablets!

Tablets are an awesome thing these days, so we decided to outsource Epsilon for tablets as well
Since it works for Android mobiles, it works awesome for Android tablets, and since it works great for iPhone it works awesome for the iPad!

Some page features

This page is powered by our very own Epsilon framework! The Epsilon framework is nothing more than a platform made from HTML5, CSS3 and JavaScript, that is the backbone of our templates! you do not have to learn a new programming language and it's extremely well documented! We chose to build our own framework so this way we know exactly what compatibility we offer with devices and we can increase the loading speed of pages to it's highest potential!
