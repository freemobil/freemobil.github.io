Welcome!

What is this?

The drawer is a sidebar navigation template, but made to run faster, better, and smoother than any of it's previous generation brothers! Although the Troller and Troller V2 have amazing features, The Drawer is Epsilon powered, which gives it a variety of new, optimized, features!

The Drawer, is the combination of all our knowledge combined in one template, powered by our very own Epsilon framework, to give a template that works smoothly, loads fast, and has a vast variety of compatibility

What's Epsilon?

Epsilon is a framework we developed in the idea of making your work faster, easier and more stable while at the same time providing features of a desktop site. This means that the page is fully jQuery powered, with custom CSS classes, and the most important thing default HTML markup!. There are no data roles, there are no complex strings, it's the same old programing langauge you've been using, but running a very powerful set of styles that asure it works on most devices out there!

What about using it? Will it be hard?

Not at all! It's very well documented, providing a documentation for our framework, in which every feature is covered, and a documentation about the template at hand, in which all the structure is properly explained! And the best part is, we offer support! If you're having trouble understanding something, you tell us, we help you, and we'll update the documentation as well to make sure it's even better explained!
