All the icons in our pack!

To locate a class of an icon, it's best you use Google Chrome! Right click on the icon you wish to use, and select Inspect Element! For example, if you right click the first icon in the pack, you'll see it open a list, where it says "icon icon-arrow-down" the name of the icon being icon-arrow-down! This applies for all elements inside the pack which have icon using abilities!

Please note! The icons are packed into two groups! Icons with and without shadow! Some templates may not be able to use this icon shadow because of the dark background of the span! You can change the span color by editing the .icon class in framework-style.css!
